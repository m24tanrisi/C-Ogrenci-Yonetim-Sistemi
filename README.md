# C-Ogrenci-Yonetim-Sistemi
C dili ile gelistirilmis ogrenci yonetim sistemi projesi
Bu proje, C programlama dili kullanılarakgelişritilmiş bir ogrenci yonetim sistemidir.Program;öğrenci bilgilerini ekleme,listeleme ve yonetme islemlerini yapabilmektedir.

##kullandığım yapılar:
-struct yapilari
-fonksiyonlar
-header dosyaları(.h)

##dosya yapisi
-ana.c:programın ana çalıştığı dosya
-ogrenci.c:Ogrenci islemleri
-ogrenci.h:struct ve fonksiyon prototipleri

##derleme
gcc ana.c ogrenci.c -o ogrenci(C dosyalarını tek program haline getirir)
